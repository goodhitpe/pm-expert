## 4. 참고용 핵심 공식 정리 (검증 완료)

### EVM 11개 지표

| 지표 | 공식 | 해석 |
|------|------|------|
| **SV** | EV − PV | > 0: 앞섬 / < 0: 지연 |
| **CV** | EV − AC | > 0: 절감 / < 0: 초과 |
| **SPI** | EV / PV | > 1: 앞섬 / < 1: 지연 |
| **CPI** | EV / AC | > 1: 절감 / < 1: 초과 |
| **EAC** (일반) | BAC / CPI | 현재 추세 유지 |
| **EAC** (일회성) | AC + (BAC − EV) | 초기 편차만 발생 |
| **EAC** (복합) | AC + (BAC − EV) / (CPI × SPI) | 일정+원가 모두 영향 |
| **ETC** | EAC − AC | 남은 필요 원가 |
| **VAC** | BAC − EAC | 완료 시 예산 편차 |
| **TCPI** (BAC) | (BAC − EV) / (BAC − AC) | 원래 목표 달성 CPI |
| **TCPI** (EAC) | (BAC − EV) / (EAC − AC) | 조정 목표 달성 CPI |

### PERT/CPM

| 공식 | 수식 |
|------|------|
| PERT 기대 시간 | $T_E = (O + 4M + P) / 6$ |
| PERT 표준편차 | $SD = (P - O) / 6$ |
| EF | ES + Duration |
| LS | LF − Duration |
| TF (총 여유) | LS − ES (또는 LF − EF) |
| FF (자유 여유) | 후행 최소 ES − EF |

### 리스크

| 항목 | 내용 |
|------|------|
| EMV | P × I |
| 위협 4전략 | 회피 · 전가 · 완화 · 수용 |
| 기회 4전략 | 활용 · 공유 · 증대 · 수용 |

## 5. 종합 의견

전체 16일차 강의자료의 PM 관련 공식·용어·프로세스는 PMBOK 6판 기준으로 **정확하고 체계적**으로 작성되어 있습니다.  

발견된 오류는 **1건**으로, Day01 PMBOK 발행 이력에서 4판에 이해관계자 관리 영역이 추가됐다고 기술한 부분이며, 실제로는 5판(2013)에서 추가된 것을 **즉시 수정 완료**했습니다.  

EVM 11개 지표, CPM/PERT 수식, 리스크 대응 전략, 계약 유형, 의사소통 채널 공식 등 PMP 시험에 자주 출제되는 핵심 공식들은 모두 정확합니다.

기술 영역(Day12~16)의 OSI 7계층, CIA 트라이어드, ACID 속성, 클라우드 서비스 모델 등도 업계 표준에 부합합니다.
