💰 원가의 유형

구분 유형 A 유형 B
직접비 vs 간접비 직접비: 프로젝트에 직접 귀속
(개발인건비, 장비구매비)		 간접비: 여러 프로젝트 공유
(사무실 임대료, 관리비)
고정비 vs 변동비 고정비: 수량 무관 일정
(사무실 임대, 라이센스)		 변동비: 수량에 비례
(재료비, 시간당 인건비)
매몰비용 (Sunk Cost)

이미 지출되어 회수 불가능한 비용. 의사결정에 영향을 미치면 안 됨!
예: "이미 3억 투자했으니 계속해야 한다" → ❌ 잘못된 판단

기회비용 (Opportunity Cost)

선택하지 않은 대안의 가치.
예: A프로젝트(수익 5억) vs B프로젝트(수익 3억) → A 선택 시 기회비용 = 3억

📈 EVM (획득가치 관리)

EVM이란?

범위 + 일정 + 원가를 통합하여 프로젝트 성과를 객관적으로 측정하는 기법

PV
계획가치 (Planned Value)
현 시점까지 계획된 작업의 예산
EV
획득가치 (Earned Value)
현 시점까지 실제 완료한 작업의 예산 가치
AC
실제원가 (Actual Cost)
현 시점까지 실제 지출한 비용

📊 EVM 성과 지표

지표 공식 > 0 (또는 > 1) < 0 (또는 < 1)
SV (일정 차이) EV - PV ✅ 일정 앞섬 ❌ 일정 지연
CV (원가 차이) EV - AC ✅ 예산 절감 ❌ 예산 초과
SPI (일정 성과 지수) EV / PV ✅ 일정 앞섬 ❌ 일정 지연
CPI (원가 성과 지수) EV / AC ✅ 예산 절감 ❌ 예산 초과
💡 암기 팁

SV/CV: EV가 항상 먼저! (EV-PV, EV-AC)
SPI/CPI: EV가 항상 분자! (EV/PV, EV/AC)
값이 양수(+) / 1보다 크면 → 좋은 상태 😊

🔮 완료 시 예측 (EAC)

공식 적용 상황 EAC 계산
EAC = BAC / CPI 현재 원가 추세가 지속될 때 가장 자주 사용
EAC = AC + (BAC - EV) 비전형적 차이 (일회성 문제) 남은 작업은 계획대로
EAC = AC + (BAC-EV)/(CPI×SPI) 일정과 원가 모두 고려 가장 보수적
ETC (잔여 예측) = EAC - AC
VAC (완료 시 차이) = BAC - EAC
TCPI (잔여 성과 지수) = (BAC - EV) / (BAC - AC)

🔨 실습: EVM 계산

📋 실습 시나리오

BAC = 10억원, 6개월 프로젝트, 현재 3개월차

PV = 5억, EV = 4억, AC = 6억

  1. SV, CV 계산 → 일정/원가 상태 판단
  2. SPI, CPI 계산 → 성과 지수 해석
  3. EAC 3가지 공식으로 계산
  4. 프로젝트 상태 진단 보고서 작성
정답 확인

SV = 4-5 = -1억(지연) | CV = 4-6 = -2억(초과)
SPI = 4/5 = 0.8 | CPI = 4/6 = 0.67
EAC(추세) = 10/0.67 = 약 15억원 → 심각한 예산 초과 예상!
