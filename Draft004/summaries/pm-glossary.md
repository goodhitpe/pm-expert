## 5. 원가 & EVM (획득가치 관리)

> 💡 **EVM이란?** 계획 대비 실제 진척을 돈으로 표현하는 방법. "얼마나 했는가"를 "얼마짜리를 했는가"로 측정.

| 약어           | 영문 풀네임                   | 한국어 의미                                          | 공식               | 처음 등장 |
| -------------- | ----------------------------- | ---------------------------------------------------- | ------------------ | --------- |
| **EVM**  | Earned Value Management       | 획득가치 관리                                        | —                 | Day 5     |
| **BAC**  | Budget At Completion          | 완료시 예산 (전체 계획 예산)                         | —                 | Day 5     |
| **PV**   | Planned Value                 | 계획가치 (지금까지 했어야 할 일의 예산)              | BAC × 계획%       | Day 5     |
| **EV**   | Earned Value                  | 획득가치 (지금까지 실제로 한 일의 예산)              | BAC × 실제%       | Day 5     |
| **AC**   | Actual Cost                   | 실제비용 (실제로 쓴 돈)                              | —                 | Day 5     |
| **SV**   | Schedule Variance             | 일정 편차                                            | EV - PV            | Day 5     |
| **CV**   | Cost Variance                 | 원가 편차                                            | EV - AC            | Day 5     |
| **SPI**  | Schedule Performance Index    | 일정 성과 지수                                       | EV ÷ PV           | Day 5     |
| **CPI**  | Cost Performance Index        | 원가 성과 지수                                       | EV ÷ AC           | Day 5     |
| **EAC**  | Estimate At Completion        | 완료시 산정 (최종 예상 수행비용)                     | 여러 공식 존재     | Day 5     |
| **ETC**  | Estimate To Complete          | 완료까지 산정 (남은 작업 예상 비용)                  | EAC - AC           | Day 5     |
| **VAC**  | Variance At Completion        | 완료시 편차                                          | BAC - EAC          | Day 5     |
| **TCPI** | To-Complete Performance Index | 완료 성과 지수 (목표 달성을 위해 앞으로 필요한 효율) | (BAC-EV)÷(BAC-AC) | Day 5     |

> 💡 **읽는 방법:**
>
> - SV/CV **양수(+)** = 좋다 (일정 앞섬 / 비용 절감)
> - SV/CV **음수(-)** = 나쁘다 (일정 지연 / 비용 초과)
> - SPI/CPI **1.0 이상** = 좋다 / **1.0 미만** = 나쁘다

## 💡 빠른 참조 카드

### EVM 공식 한눈에 보기

```
편차 (양수 = 좋음, 음수 = 나쁨)
  SV = EV - PV   (일정 편차)
  CV = EV - AC   (원가 편차)

지수 (1.0 이상 = 좋음, 1.0 미만 = 나쁨)
  SPI = EV ÷ PV  (일정 성과 지수)
  CPI = EV ÷ AC  (원가 성과 지수)

예측
  EAC = AC + (BAC - EV) ÷ CPI  [가장 자주 쓰임]
  ETC = EAC - AC
  VAC = BAC - EAC
  TCPI = (BAC - EV) ÷ (BAC - AC)
```

### 계약 유형 리스크 비교

```
구매자 리스크 낮음 ←─────────────────────────→ 구매자 리스크 높음
FFP         FPIF        CPFF        CPIF        T&M
고정가격                원가+수수료              시간+재료비
공급사 리스크 높음 ←─────────────────────────→ 공급사 리스크 낮음
```

### CPM 주공정 계산 순서

```
1. 네트워크 다이어그램 작성 (활동 순서 연결)
2. 전진 계산 (ES, EF) → 최조 시작/완료일
3. 후진 계산 (LS, LF) → 최지 시작/완료일
4. 여유 계산: TF = LF - EF
5. 주공정: TF = 0인 경로
```
