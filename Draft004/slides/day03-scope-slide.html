제품 범위 vs 프로젝트 범위

제품 범위 (Product Scope)

"무엇을 만들 것인가 (What)"
최종 인도물의 특성과 기능.
고객/사용자 관점.

  • 로그인 기능
  • 결제 모듈
  • 보고서 출력 기능
  • 실시간 알림

📄 요구사항 명세서, 설계 문서로 표현

프로젝트 범위 (Project Scope)

제품을 인도하기 위해 수행할 작업.
프로젝트 팀 관점.

  • 요구사항 분석 (2주)
  • UI/UX 설계 (2주)
  • 백엔드 개발 (6주)
  • 통합 테스트 (2주)

MoSCoW 우선순위 & RTM

MoSCoW 기법

Must have

필수. 없으면 프로젝트 실패.
예: 로그인, 결제 기능

Should have

중요하나 대안 가능.
예: 실시간 동기화

Could have

있으면 좋음. 여유 시 구현.
예: 다크 모드

Won't have (this time)

이번엔 제외. 차기 버전.
예: AR 기능

WBS 작성

⏱ 1.5시간

WBS (Work Breakdown Structure)

🌳 WBS란?
프로젝트의 인도물과 작업을 더 작고 관리 가능한 구성요소로 계층적으로 분해한 그림
100% 규칙
부모 WBS = 모든 자식의 합
• 누락 없이 (일부 작업이 WBS 밖에 있으면 ❌)
• 중복 없이 (같은 작업이 두 곳에 있으면 ❌)
Work Package (작업 패키지)
WBS의 최하위 구성요소.
• 1명이 담당 가능한 크기
• 비용·일정 산정 가능
• 책임자 배정 가능
• Work Package ≠ 활동 (Activity)
활동 = 작업 패키지를 실행하기 위한 세부 작업
📋 WBS 사전 (WBS Dictionary)
각 Work Package의 상세 설명
→ 코드번호, 작업설명, 담당자, 기간, 비용, 인수기준

범위 확인 vs 품질 통제 & Scope Creep

범위 확인 (Validate Scope)

완성된 인도물의 공식 인수

  • 고객/스폰서가 서명
  • 인도물 = 범위 기술서 충족 여부
  • 감시/통제 프로세스
  • 인수 기준 기반 판단

질문: "이걸 납품받겠습니까?"

품질 통제 (Control Quality)

인도물이 품질 기준 충족 여부 확인

  • QA 팀이 검사
  • 인도물 = 품질 기준 충족 여부
  • 감시/통제 프로세스
  • 결함 발견 → 수정 → 재검사

질문: "제대로 만들었습니까?"

⚠️ Scope Creep — 원인과 예방

주요 원인
❌ 불명확한 초기 요구사항
❌ 요구사항 변경 통제 부재
❌ 고객 추가 요구 무분별 수용
❌ PM의 거절 능력 부족
예방 방법
✅ 명확한 범위 기술서 + 제외사항
✅ 변경 통제 프로세스 (CCB)
✅ RTM으로 요구사항 추적
✅ 이해관계자 지속 소통

학습 확인 퀴즈

Q1. 제품 범위와 프로젝트 범위의 차이를 설명하고 각각의 예시를 2개씩 드시오.

▶ 모범 답안

제품 범위: 최종 인도물의 특성·기능 ("무엇을"). 예: 로그인 기능, 결제 모듈
프로젝트 범위: 제품을 인도하기 위한 작업 ("어떻게"). 예: 요구사항 분석, 통합 테스트
핵심: 제품 범위가 명확해야 프로젝트 범위 정의 가능. 제품 범위 변경 → 프로젝트 범위 반드시 변경.

Q2. WBS의 100% 규칙이란 무엇이며 왜 중요한가?

▶ 모범 답안

WBS의 최상위 인도물은 모든 하위 작업의 합과 같아야 한다는 원칙.
누락 없이: 모든 범위가 WBS에 포함 (빠진 작업 = 예산·일정 외 작업)
중복 없이: 같은 작업이 두 곳에 있으면 이중 비용·책임 혼란
결과: 범위 기준선의 완전성 보장, 비용·일정 산정의 정확성 확보

Q3. 범위 확인(Validate Scope)과 품질 통제(Control Quality)의 차이를 설명하시오.

▶ 모범 답안

범위 확인: "이걸 납품받겠습니까?" — 완성된 인도물을 고객이 공식 인수하는 프로세스. 인도물이 범위 기술서의 요구사항을 충족하는지 판단.
품질 통제: "제대로 만들었습니까?" — 인도물이 품질 기준을 충족하는지 내부에서 검사. 결함 발견 시 수정 후 재검사.
순서: 품질 통제 → 범위 확인 (품질 통과 후 고객 인수)

Q4. Scope Creep의 원인 3가지와 예방 방법 3가지를 설명하시오.
